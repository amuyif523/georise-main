# GEORISE
AI-Driven Geospatial Platform for Incident Management and Multi-Agency Dispatch System for Addis Ababa City

### Tech Stack
- Frontend: React + TypeScript + Tailwind CSS
- Backend: Node.js + Express + TypeScript
- AI: Python + FastAPI
- Database: PostgreSQL + PostGIS
- DevOps: Docker & Docker Compose

### Folders
| Folder | Description |
|--------|-------------|
| frontend | UI & dashboards |
| backend | Core REST API & auth |
| ai-service | AI classification service |
| infra | Docker DB & infra scripts |
| docs | Documentation, UML, Proposal |
